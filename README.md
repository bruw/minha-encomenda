# Minha Encomenda

App para rastreamento de encomendas transportadas pelos correios. 

Desenvolvido na disciplina de programação para dispositivos móveis (PDM) do curso de TSI da UTFPR - Guarapuava

## Tecnologias

+ Node.js
+ Yarn 
+ Expo
+ React Native

## Ferramentas

+ Visual Studio Code
+ SQLite

## Versão Corrente

1.0.0 - Release de 31/08/2021
